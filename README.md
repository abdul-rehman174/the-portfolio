# the-portfolio

Personal portfolio of **Abdul Rehman** — Backend Developer specializing in
Django, FastAPI, and scalable multi-tenant SaaS systems.

🌐 Live: [madebyabdul.me](https://madebyabdul.me)

## Tech stack

- **HTML / CSS / JavaScript** — no build step, fully static
- **[GSAP](https://gsap.com/) + ScrollTrigger** — scroll and reveal animations
- **[Lenis](https://github.com/darkroomengineering/lenis)** — smooth scrolling
- **Google Fonts** — Space Grotesk + Inter

## Features

- Animated gradient mesh background that reacts to the cursor
- Smooth scroll, split-text reveals, animated counters
- Custom cursor and magnetic buttons
- Fully responsive, accessible fallbacks, and reduced-motion support
- Hardened with a Content Security Policy and SRI on all CDN scripts

## Run locally

```bash
python3 -m http.server 5500
# then open http://localhost:5500
```

## Deploy

Hosted on GitHub Pages with a custom domain (see `CNAME`). Any push to the
default branch publishes automatically.
